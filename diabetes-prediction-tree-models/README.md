# 🩺 Explainable Diabetes Prediction Using Tree-Based Models

## 🎯 Problem
The goal of this project was to predict whether a patient has diabetes
based on diagnostic health measurements using interpretable machine learning models.

## 📊 Dataset
The Pima Indians Diabetes dataset was used for this project.

Original dataset source:
https://archive.ics.uci.edu/ml/datasets/Pima+Indians+Diabetes

CSV file used in this project:
https://gist.githubusercontent.com/dmpe/bfe07a29c7fc1e3a70d0522956d8e4a9/raw/pima-indians-diabetes.csv

## ⚙️ Approach
The project includes:
- Exploratory Data Analysis (EDA)
- Correlation analysis
- Data preprocessing and feature scaling
- Principal Component Analysis (PCA)
- Supervised classification using:
  - Decision Trees
  - Random Forests
- Hyperparameter tuning using 10-fold cross-validation

## 📈 Results
The Random Forest model achieved better generalization performance
than a single Decision Tree classifier.

The project also includes:
- Confusion matrix analysis
- Precision, recall, and F1-score evaluation
- PCA visualization and interpretation
