Global Tuberculosis Burden: Trends, Regional Inequality, and Country-Level Risk

Author

Zohreh Sharif Kazemi

Overview

Tuberculosis (TB) is a contagious disease usually caused by Mycobacterium tuberculosis (MTB) bacteria, most commonly affecting the lungs. While many people infected with TB do not develop symptoms (latent TB), a smaller proportion develops active disease, which can be serious and even fatal if left untreated. In the 19th century, tuberculosis became one of the leading causes of death in Europe and North America. Although antibiotics greatly reduced TB deaths in the 20th century, TB remains a major global health problem today, especially in low- and middle-income countries.

This project explores global tuberculosis burden using country-level data from the World Health Organization (WHO) covering the period 2000–2023. It examines long-term trends in TB incidence and mortality, compares patterns across regions, and identifies countries with relatively high or low TB burden.

The main question is:

How has the global TB burden changed over time and across regions, and how does it vary across countries in the most recent data?

To address this question, TB incidence and mortality trends are visualized over time and compared across WHO regions. In addition, countries are grouped into different TB burden levels using k-means clustering based on the most recent data.

Data

This analysis uses the WHO TB Burden dataset, which provides country-level estimates of TB incidence, mortality, and population from 2000 to 2023.

The WHO TB Burden dataset contains 5,117 observations. Each row represents one country in one specific year. The full dataset includes 18 variables, although only a subset is required for this analysis. The following table provides a detailed description of the variables used.

variable class description
country character Country or territory name
g_whoregion character Country or territory region
iso3 character ISO 3-character country/territory code
year integer Year of observation (2000-2023)
e_inc_100k double Estimated incidence (all forms) per 100,000 population
e_mort_100k double Estimated mortality of TB cases (all forms) per 100,000 population

More information about the dataset and its source is available here.

Methods

To address the research question, the data are first cleaned and prepared for analysis. Trends in TB incidence (e_inc_100k) and mortality (e_mort_100k) are visualized over time using smoothed trend lines, which help highlight overall patterns in a large dataset.

The results are presented using a two-panel layout created with the patchwork library. Panel A shows the global trend from 2000 to 2023, while Panel B compares trends across regions using facet_wrap(~g_whoregion).

To examine country-level differences in the most recent data, k-means clustering is applied to estimated TB incidence and mortality. Countries are grouped into five clusters representing different levels of TB burden, and the results are displayed on a world map using geom_sf().

Applying K-means Method to Cluster the Data

# Elbow plot for choosing number of clusters
calc_withinss <- function(data, centers) {
  km_fit <- select(data, where(is.numeric)) |>
    kmeans(centers = centers, nstart = 10)
  km_fit$tot.withinss
}

tibble(centers = 1:15) |>
  mutate(
    within_sum_squares = map_dbl(
      centers, ~calc_withinss(filter (who_tb_data, year == 2023), .x)
    )
  ) |>
  ggplot() +
  aes(centers, within_sum_squares) +
  geom_point() +
  geom_line()+
  labs(
  x = "Number of Clusters (k)",
  y = "Within-Cluster Sum of Squares"
)

The elbow plot suggests using 4–5 clusters. Therefore, k = 5 is selected for the clustering analysis.

set.seed(123)
km_fit <- who_tb_data |>
  filter (year == 2023)|>
  select(e_inc_100k, e_mort_100k)|>
  scale()|>
  kmeans(centers = 5, nstart = 50)

# Order clusters from lowest to highest TB burden
cluster_labels <- tidy(km_fit) |>
  mutate(distance = e_inc_100k + e_mort_100k) |> # Clusters are ordered based on combined incidence and mortality levels.
  arrange(distance) |>
  mutate(
    cluster_order = c("Lowest TB Burden", "Low TB Burden", "Moderate TB Burden", "High TB Burden", "Highest TB Burden"),
    cluster_order = fct_relevel(cluster_order, "Lowest TB Burden", "Low TB Burden", "Moderate TB Burden", "High TB Burden", "Highest TB Burden")
    )

cluster_labels_tojoin <- select(cluster_labels, cluster, cluster_order) 

# Prepare scaled data for visualizing clusters
plot_data <- who_tb_data |>
  filter (year == 2023)|>
  mutate(
    # Convert scaled values to numeric vectors
    e_inc_100k  = as.numeric(scale(e_inc_100k)),
    e_mort_100k = as.numeric(scale(e_mort_100k))
  )

km_fit |>
  # combine with original data, scaled on two columns e_inc_100k and e_mort_100k
  augment(plot_data) |>
  left_join(cluster_labels_tojoin, by = c(".cluster" = "cluster")) |>
  ggplot() +
  aes(x = e_inc_100k, y = e_mort_100k) +
  geom_point( # points representing original data
    aes(color = cluster_order)
  ) +
  geom_point( # points representing centroids
    data = cluster_labels,
    aes(x = e_inc_100k, y = e_mort_100k, fill = cluster_order),
    shape = 21, color = "black", size = 4
  ) +
  labs(
  x = "Scaled TB Incidence",
  y = "Scaled TB Mortality",
  fill = "TB Burden Level"
  ) +
  guides(color = "none")

Discussion

Based on Figure 1 panel A, there is a clear global decrease in the number of incident TB cases after 2000. The plot also shows a decrease in mortality of TB cases, but this decrease is less noticeable compared to incidence.

Across the regions (Figure 1 panel B), the Americas do not show any visible change in incidence and mortality and both stay in low levels (especially mortality, which is close to zero). This suggests that TB is well controlled in this region. In Europe, the number of incident TB cases decreases over time, while mortality remains very low after 2000. The Eastern Mediterranean region also shows a decrease in incidence, and TB mortality approaches zero in recent years. In the Western Pacific region, there is no clear change in incidence or mortality after 2000, but the level of TB incidence remains higher compared to the Americas and Europe.

The regions that still seem to face challenges in controlling TB are South-East Asia and Africa. In South-East Asia, TB incidence decreases from 2005 to 2019 but shows a slight increase after 2020. TB mortality decreases between 2000 and 2015 but shows no change after 2015. In Africa, both incidence and mortality decrease over time, but both levels are still high. Overall, in 2023, South-East Asia and Africa show similar levels of TB burden, which are higher compared to other regions

Figure 2 shows the global distribution of TB burden in 2023. The map is interactive, allowing users to explore country-level TB burden by hovering over each country. Five TB burden levels are considered. Although the majority of countries fall into the lowest, low, and moderate TB burden, there are still some countries with high and highest TB burden. As expected from Figure 1, countries with high TB burden are located in Africa and South-East Asia. Four African countries with the highest TB burden include the Central African Republic, Gabon, Lesotho, and Guinea-Bissau.
